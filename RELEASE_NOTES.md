# Release Notes - v1.0

## Features
- Merge half-page invoices into A4 pages (Top/Bottom).
- Supports drag-and-drop file organization.
- Intelligent invoice detection (Number, Date).
- Handles nested ZIP/RAR archives (up to 3 levels).
- Duplicate invoice detection and conflict blocking.
- Real-time statistics.
- Multi-language support (English/Chinese).
- Direct print preview using system viewer.

## Compatibility
- **Supported OS**: Windows 10, Windows 11 (64-bit).
- **Not Supported**: Windows 7 (Due to Python 3.11 dependency).
  - *Note for Developers*: To build a Windows 7 compatible version, you must use Python 3.8 and rebuild the application.

## Installation
1. Download `PDFInvoiceMerger.exe`.
2. Run directly (Portable, no installation required).
