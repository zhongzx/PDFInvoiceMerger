# PDFInvoiceMerger 功能更新文档 (2026-01-10)

本文档记录了 PDFInvoiceMerger 的最新功能改进、界面优化以及跨平台支持的详细说明。

## 1. 界面布局与交互优化
- **现代三栏布局**：
    - 引入 `QSplitter` 实现可调节的三栏布局。
    - **左侧列表**：展示识别成功的发票和行程单。
    - **中间区域**：核心操作按钮（合并、保存、打印等）及大尺寸 PDF 预览区。
    - **右侧列表**：隐藏式“识别失败”列表，仅在有无法识别的页面时自动显示。
- **全局拖拽支持**：
    - 界面任何区域均可接受文件拖拽（支持 PDF 及 ZIP/RAR/7z 压缩包）。
- **右键菜单管理**：
    - 移除列表下方冗余按钮，改用右键菜单进行“删除所选”或“清空列表”操作。
- **列表显示优化**：
    - 增加字体大小和行间距，提升阅读舒适度。
    - 列表项直接显示提取的关键信息（如：发票 YYYY-MM-DD #########）。

## 2. 智能识别与分类逻辑
- **行程单自动识别**：
    - 识别关键词：“收费公路通行费电子票据汇总单”。
    - **关键信息提取**：
        - 开票申请日期（右上角）：格式化为 `YYYY-MM-DD`。
        - 汇总单号（左上角）：支持变长数字提取。
- **发票自动识别**：
    - 识别关键词：“发票”或 “Invoice”。
    - 提取开票日期及发票号码。
- **自动分离逻辑**：
    - 识别成功的页面进入主列表，识别失败的页面进入右侧列表并显示前10个字符以供辨认。

## 3. 自动化合并与缩放算法
- **全局自动排序**：
    - 所有待处理页面统一按 **日期（升序） -> 编号（升序）** 进行全局排序，确保合并结果逻辑清晰。
- **行程单智能缩放**：
    - 优先尝试按 **80%** 比例缩小以适应半页 A4（A5 大小）。
    - 若缩小后仍无法适应半页，则保持原尺寸并**独占一页 A4 且居中显示**。
- **重复检测**：
    - 自动检测相同发票号码/行程单号，红色背景高亮冲突项，蓝色背景显示已选中的唯一项。

## 4. 跨平台深度适配
- **原生快捷键支持**：
    - **Windows**：支持 `Alt + 字母` 的 Access Key（带下划线提示）。
    - **macOS**：支持 `Cmd + 字母` 组合键，符合 Apple 人机交互指南。
    - **Linux**：支持 `Alt + 字母`（显示为 `(_P)` 风格）。
- **系统级调用**：
    - **打印预览**：根据操作系统自动选择最优调用方式（macOS: `open`, Windows: `os.startfile`, Linux: `xdg-open`）。
- **运行脚本**：
    - 提供 `run_merger.sh` 脚本，自动处理 macOS 环境下的虚拟环境创建、依赖安装及程序启动。
